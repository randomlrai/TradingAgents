# TradingAgents: Multi-Agents LLM Financial Trading Framework

> **Personal fork** — I'm using this primarily to experiment with US tech stocks (NVDA, MSFT, AAPL) and to learn how multi-agent LLM pipelines work. Notes and tweaks may appear throughout.
